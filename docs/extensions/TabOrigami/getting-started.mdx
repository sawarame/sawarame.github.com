# Getting Started

TabOrigamiを使い始めるための基本的な手順を説明します。

## 1. APIキーの取得

TabOrigamiは、「BYOK (Bring Your Own Key)」形式を採用しているため、ユーザー自身がGoogle Gemini APIのAPIキーを取得して設定する必要があります。以下の手順でAPIキーを取得してください。

1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスして、Googleアカウントでログインします。
2. 「APIキー」から「APIキーを作成」ボタンをクリックして、新しいAPIキーを生成します。
3. 任意のキー名を入力し、プロジェクトを選択してください。プロジェクトは既存のものを選んでも、新規に作成しても構いません。
4. 「キーを作成」ボタンをクリックしてAPIキーが生成されたら、**キーをコピー**しておきます。

## 2. APIキーの設定

取得したAPIキーをTabOrigamiに設定する手順は以下の通りです。

1. Chromeの拡張機能「TabOrigami」のアイコンをクリックしてTabOrigamiのポップアップを開きます。
2. 画面右上の設定アイコン⚙️をクリックし **APIキー** を登録します。
3. 必要に応じて、言語（日本語 / English）や使用するGeminiモデルを変更してください。

## 2. タブを整理する

設定が完了したら、実際にタブを整理してみましょう。

1. ポップアップ画面で、現在の作業状況に合った「折り方（整理スタイル）」を選択します。
2. AIがタブを解析し、提案内容がリスト（プレビュー）として表示されます。
3. 提案内容を確認し、ドラッグ＆ドロップでタブを移動させたり、チェックボックスを外して整理から除外したりすることができます。
4. タブグループ名やカラーも自由に編集可能です。
5. **「実行する」** ボタンをクリックすると、Chromeのタブが実際のグループとして整理されます。
