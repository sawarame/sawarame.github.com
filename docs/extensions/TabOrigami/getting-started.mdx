# Getting Started

TabOrigamiを使い始めるための基本的な手順を説明します。

## 1. APIキーの設定

TabOrigamiは、あなた自身のGemini APIキーを利用する「BYOK (Bring Your Own Key)」形式を採用しています。

1. 拡張機能のアイコンをクリックしてTabOrigamiのポップアップを開きます。
2. 画面下部の設定アイコンをクリックし、ご自身の **Gemini API キー** を登録します（必須）。
3. 必要に応じて、言語（日本語 / English）や使用するGeminiモデル（例: gemini-2.5-flash）を選択します。

## 2. タブを整理する

設定が完了したら、実際にタブを整理してみましょう。

1. ポップアップ画面で、現在の作業状況に合った「折り方（整理スタイル）」を選択します。
2. AIがタブを解析し、提案内容がリスト（プレビュー）として表示されます。
3. 提案内容を確認し、ドラッグ＆ドロップでタブを移動させたり、チェックボックスで対象タブを微調整したりします。
4. **「実行する」** ボタンをクリックすると、Chromeのタブが実際のグループとして整理されます。スマートクリーンアップ機能により、不要な「新しいタブ」なども自動で閉じられます。

## 3. 整理前に戻す（履歴機能）

整理を実行した結果が気に入らなかったり、以前の状態に戻したくなったりした場合は、以下の手順で復元できます。

1. ポップアップ画面上部の「履歴」アイコンをクリックします。
2. 過去最大10件までの実行前状態がリスト表示されるので、戻したい時点の履歴を選択します。
3. その時点のタブ構成に復元されます。
