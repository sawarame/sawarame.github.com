# TabOrigami プライバシーポリシー

TabOrigami（以下、「本拡張機能」といいます）は、ユーザーの皆様のプライバシーを最優先に考え、安全に利用できる環境を提供することに努めています。本プライバシーポリシーでは、本拡張機能における個人情報やデータの取り扱いについて説明します。

## 1. 取得する情報とその利用目的

本拡張機能は、タブの整理を行うために以下の情報を利用しますが、**これらの情報はすべてユーザーの端末内またはGoogleの公式Gemini APIとの通信のみで使用され、開発者を含む第三者の外部サーバーに送信・保存されることはありません**。

* **開いているタブの情報（タイトル、URL、ページ内のテキスト情報）**:
  AIによる高度な分類・整理を行うために、これらの情報を取得し、Gemini APIに送信します。
* **Gemini API キー**:
  本拡張機能は「Bring Your Own Key（BYOK）」方式を採用しています。入力されたAPIキーは端末内のローカルストレージ（ブラウザの同期ストレージ）にのみ安全に保存され、Googleの公式Gemini APIとの通信にのみ利用されます。

## 2. データの外部送信について

本拡張機能は、タブの整理機能を実行する際、Google LLCが提供する「Gemini API」に対して、取得したタブ情報とユーザーが指定したAPIキーを直接送信します。

* 送信されたデータは、GoogleのAPI利用規約およびプライバシーポリシーに従って処理されます。
* **開発者（Sawarame）の管理するサーバーや、その他の第三者サーバーにデータが送信されることは一切ありません。**

## 3. データの保存

* ユーザーの設定（APIキー、選択した言語、モデル名などの設定情報）および、過去のタブ整理の履歴（最大10件）は、お使いのブラウザのローカル環境（`chrome.storage.local` 等）に保存されます。
* これらのデータは、本拡張機能をアンインストールすることで完全に削除されます。

## 4. セキュリティ

本拡張機能は、ブラウザ拡張機能の最新のセキュリティ基準である「Manifest V3」に準拠して開発されています。不要な権限の要求を最小限に抑え、安全なデータ通信（HTTPS）を通じてAPIリクエストを行います。

## 5. プライバシーポリシーの変更

本プライバシーポリシーは、機能の追加や法令の変更などに伴い、予告なく改定される場合があります。重要な変更がある場合は、拡張機能のアップデートノートや公式ページ等でお知らせします。

## 6. お問い合わせ

本拡張機能に関する不具合報告、ご意見、ご要望、またはプライバシーに関するお問い合わせは、[GitHub Issues](https://github.com/sawarame/TabOrigami/issues) にて受け付けております。
