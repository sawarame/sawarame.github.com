# Feature Overview

TabOrigamiに搭載されている多彩な整理機能とカスタマイズ機能をご紹介します。

## 柔軟な整理スタイル（折り方）

現在の作業状況に合わせて、最適なカテゴリ分類を選択できます。

- **おまかせ**: AIが最適なカテゴリを自動判断して整理します。
- **タスク集中**: 進行中の作業やプロジェクトごとにタブをまとめます。
- **仕事とプライベート**: ビジネス関連と個人的なタブを明確に分離します。
- **断捨離**: 不要なタブを整理しやすくし、ブラウザをスッキリさせます。
- **カスタム（独自プロンプト）**: 自由なテキストでAIに追加指示を与え、自分好みのグループ化を実現します。

## グループ名の変更と色指定

AIが提案したグループ名や色を、実行前に自由に変更できます。

- 整理前のプレビュー画面で、提案されたグループ名をクリックして直接編集できます。
- カスタムカラーピッカーを使用して、Chrome上で表示されるグループの色を自分好みに選択できます。
- 設定した名前や色は、実行後の実際のChromeタブグループにそのまま反映されます。

## 固定タブの除外

常に開いておく重要なタブ（ピン留めしたタブ）を保護する機能です。
設定を有効にすることで、ピン留めされたタブは整理の対象外となり、配置が維持されます。

## スマートクリーンアップ機能

タブを整理する際、不要な「新しいタブ」などを自動的に検出して閉じます。
また、同一ドメインのタブは関連性が高いと判断し、効果的にまとめて整理します。
