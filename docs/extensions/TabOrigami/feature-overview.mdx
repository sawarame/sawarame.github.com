# Feature Overview

TabOrigamiに搭載されている整理機能とカスタマイズ機能をご紹介します。

## 柔軟な整理スタイル（折り方）

現在の作業状況に合わせて、最適なカテゴリ分類を選択できます。

- **おまかせ**: AIが最適なカテゴリを自動判断して整理します。
- **タスク集中**: 進行中の作業やプロジェクトごとにタブをまとめます。
- **仕事とプライベート**: ビジネス関連と個人的なタブを明確に分離します。
- **断捨離**: 不要なタブ削除します。

## 独自プロンプト機能

独自プロンプトを入力して、AIに対して自由なテキストで追加指示を与えることができます。これにより、自分好みのグループ化を実現できます。

1. ポップアップの「独自プロンプトを追加」をクリックします。
2. テキスト入力欄に、AIに対して与えたい指示を自由に入力し、「保存」ボタンをクリックします。
3. TOP画面に戻って、いずれかの整理スタイルを選択して実行すると、独自プロンプトの指示も反映されたグループ化が提案されます。

:::tip
独自プロンプトには複数の指示を同時に与えることも可能です。例えば、以下のような指示を入力できます。
```
- 同じURLのページの場合は、一つだけ残してあとは削除してください。
- タブグループ名は絵文字にしてください。
```
:::

## 4. 整理前に戻す（履歴機能）

整理を実行した結果が気に入らなかったり、以前の状態に戻したくなったりした場合は、以下の手順で復元できます。

1. ポップアップの「整理前に戻す」をクリックします。
2. 過去最大10件までの実行前状態がリスト表示されるので、戻したい時点の履歴を選択します。
3. その時点のタブ構成に復元されます。

## 固定タブの除外

常に開いておく重要なタブ（ピン留めしたタブ）を保護する機能です。
設定を有効にすることで、ピン留めされたタブは整理の対象外となり、配置が維持されます。

オプション画面で「固定タブを整理から除外する」をオンにしてください。これにより、ピン留めされたタブは常に整理の対象外となります。
