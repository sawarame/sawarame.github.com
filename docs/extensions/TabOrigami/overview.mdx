TabOrigamiは、Gemini AIの力を活用して、ブラウザのタブをスマートに整理する拡張機能です。

開きすぎて収集がつかなくなった大量のタブを、AIが文脈に基づいて賢く分類・グループ化します。仕事とプライベートのタブを分けたり、プロジェクトごとにまとめたり、不要なタブを整理したりすることができます。
