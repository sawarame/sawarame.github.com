## 使い方

1. ファイルをドラッグ＆ドロップするか、クリックしてファイルを選択してください。コピーしたテキストやファイルを直接ペースト（Ctrl+V / Cmd+V）することも可能です。
2. 自動的に文字コードが判別され、内容のプレビューが表示されます。もしプレビューが文字化けしている場合は、「現在の文字コード」を手動で変更してみてください。
3. 変換後の文字コードを選択し、「変換してダウンロード」ボタンを押すと、新しい文字コードで保存されます。

### 対応形式
- **文字コード:** UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP (JIS), UTF-16, ASCII など
- **改行コード:** LF (Unix/macOS), CRLF (Windows), CR (Classic Mac)

:::info セキュリティについて
🔒 処理はすべてブラウザ内で行われます。ファイルの内容や改行コードがサーバーに送信されることはありません。
:::
