## 使い方

1. パスワードの長さや使用する文字の種類（記号、大文字、小文字など）を指定します。
2. 設定を変更すると、自動的に新しいパスワードが生成されます。
3. 「コピー」ボタンで個別にコピーするか、「保存」ボタンで生成されたリストをテキストファイルとしてダウンロードできます。
4. パスワードの作成設定はURLに反映されますので、ブックマークしておくことで次回以降も同じ条件でパスワードを素早く生成できます。

:::info セキュリティについて
🔒 生成処理はすべてご利用のブラウザ内で行われます。パスワードや設定内容がサーバーに送信されることはありません。
:::
