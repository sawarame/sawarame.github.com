## 使い方

1. 軽量化したい画像ファイルをドラッグ＆ドロップ、またはクリックして選択します（複数選択対応）。クリップボードからの貼り付け（Ctrl+V）にも対応しています。
2. 必要に応じて、各画像の「クロップ」ボタンから画像の切り抜きを行えます。SNS向けのプリセット比率も用意しています。
3. 「最適化設定」から、出力フォーマット（WebP推奨）、最大幅、画質などを調整します。JPEG形式の場合はEXIF情報の保持を選択できます。
4. 「最適化を開始」をクリックすると、ブラウザ内で変換処理が実行されます。
5. 処理完了後、「比較」ボタンで圧縮前後の画質を確認し、問題なければ「ダウンロード」から画像を保存してください。複数枚ある場合はZIP形式でまとめてダウンロードされます。

:::info プライバシーとセキュリティ
🔒 画像の軽量化やクロップ処理はすべてお使いの端末（ブラウザ内）で実行され、外部サーバーにデータが送信されることは一切ありません。機密性の高い画像でも安心してご利用いただけます。
:::
