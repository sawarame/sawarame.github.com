## 使い方

1. 比較したい二つの日付を入力します。
2. 日付フォーマットは、スラッシュ区切り（2026/01/01）や日本語（2026年1月1日）、Unixタイムスタンプなど幅広く対応しています。
3. カレンダーアイコンをクリックすると、日付ピッカーから選択することも可能です。
4. 「常に現在時刻と比較」にチェックを入れると、1つ目の日付が自動的に現在の時刻に固定され、リアルタイムで差分が更新されます。
5. 差分結果は「日時分秒」形式と「総秒数」の両方で表示され、コピーボタンでクリップボードに保存できます。

:::info プライバシーについて
🔒 入力した日付データはすべてブラウザ内で処理され、サーバーに送信されることはありません。
:::
