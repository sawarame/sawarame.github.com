## 表示される情報

- **ネットワーク**: 現在のグローバルIPアドレスを表示します。
- **画面サイズ**:
  - **物理解像度**: `window.screen` から取得した画面全体のピクセル数です。
  - **ウィンドウサイズ**: ブラウザの表示領域（`window.innerWidth/Height`）のサイズです。
- **ハードウェア**:
  - **論理コア数**: デバイスの CPU コア数を表示します。
  - **メモリ容量**: およその RAM 容量を表示します（ブラウザのプライバシー保護により制限される場合があります）。
- **ユーザーエージェント**: ブラウザから送信される識別文字列を表示します。

:::info プライバシーについて
🔒 IPアドレスの取得を除き、すべての情報はブラウザ内で直接取得されます。取得したデバイス情報がサーバーに保存されたり、個人が特定されることはありません。
:::
