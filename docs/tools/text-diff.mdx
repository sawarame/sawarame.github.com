## 使い方

1. ファイルをドラッグ＆ドロップするか、クリックしてファイルを選択してください。コピーしたテキストやファイルを直接ペースト（Ctrl+V / Cmd+V）して追加することも可能です。
2. アップロードされたファイルが一覧表示されます。
3. **「左 (元)」** と **「右 (先)」** のラジオボタンを選択して、比較したい2つのファイルを指定してください。
4. 画面下部に差分が表示されます。
5. **「2画面表示（Split）」** と **「1画面表示（Unified）」** を切り替えて、好みの形式で確認できます。
6. **「空白を無視」** オプションを有効にすると、インデントやスペースのみの違いを除外して比較できます。

:::info セキュリティについて
🔒 処理はすべてブラウザ内で行われます。ファイルの内容がサーバーに送信されることはありません。
:::
