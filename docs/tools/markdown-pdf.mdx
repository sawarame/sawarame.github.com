## 使い方

1. **マークダウンの入力**:
   * **ファイル読み込み**: `.md` ファイルを「ファイルを選択」ボタンから選択するか、ドラッグ＆ドロップして読み込みます。
   * **テキスト貼り付け**: テキストエリアにマークダウンテキストを直接貼り付け、または入力します。
2. **オプションの設定**:
   * 出力設定パネルから、フォントサイズ、余白（マージン）、用紙サイズ（A4/Letter）、用紙の向き（縦/横）、ページ番号表示の有無を調整します。
3. **プレビューの確認**:
   * **HTMLプレビュー**: リアルタイムでマークダウンのHTML変換結果を確認できます。
   * **PDFプレビュー**: 「PDFプレビュー」タブを選択すると、指定したレイアウト設定に基づいたPDFプレビューが自動的に作成され、表示されます。
4. **ファイルのダウンロード**:
   * **HTMLダウンロード**: 「HTMLダウンロード」ボタンをクリックすると、見やすいスタイルが適用されたHTMLファイルをダウンロードできます。
   * **PDFダウンロード**: 「PDFダウンロード」ボタンをクリックすると、ブラウザ上でPDFを即時に生成し、ダウンロードできます。

:::info セキュリティについて
🔒 アップロードしたファイルや入力したテキストはすべてお使いのブラウザ内で処理されます。サーバーへ送信されることは一切ないため、機密情報が含まれるファイルでも安全にご利用いただけます。
:::
