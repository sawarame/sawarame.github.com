## How to Use

1. Drag and drop a file, or click to select a file. You can also paste (Ctrl+V / Cmd+V) copied text or files directly.
2. The character encoding will be automatically detected, and a preview of the content will be displayed. If the preview shows garbled text, try manually changing the "Current Encoding".
3. In the **"Text Information"** section, you can check the number of characters, words, lines, and the current file size.
4. When you select a target encoding or line break, the **estimated target file size** will be simulated.
5. Select the target encoding and click the "Convert and Download" button to save the file with the new encoding.

### Supported Formats
- **Encodings:** UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP (JIS), UTF-16, ASCII, etc.
- **Line Break Codes:** LF (Unix/macOS), CRLF (Windows), CR (Classic Mac)

:::info Security
🔒 All processing is done within the browser. The file content and line break codes are never sent to a server.
:::
