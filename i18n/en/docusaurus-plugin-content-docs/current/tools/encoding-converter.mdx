## How to Use

1. Drag and drop a file, or click to select a file. You can also paste (Ctrl+V / Cmd+V) copied text or files directly.
2. The character encoding will be automatically detected, and a preview of the content will be displayed. If the preview shows garbled text, try manually changing the "Current Encoding".
3. Select the target encoding and click the "Convert and Download" button to save the file with the new encoding.

### Supported Encodings
- **Input:** UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP (JIS), UTF-16, ASCII, etc.
- **Output:** UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP (JIS), UTF-16

:::info Security
🔒 All processing is done within the browser. The file content is never sent to a server.
:::
