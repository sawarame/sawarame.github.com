## How to use

1. **Enter Items**: Type your list items into the text area, one per line.
2. **Settings**: Enable "Remove after draw" to automatically delete the winner from the list for unique picks.
3. **Import / Export**: Load items from a text file or save your current list for later use.
4. **Start Draw**: Click the button to spin the roulette and pick a random item.
5. **History**: View past results in the "History" section and clear them if needed.

:::info Privacy
🔒 All data and history are stored only in your browser's local storage and are never sent to any server.
:::
