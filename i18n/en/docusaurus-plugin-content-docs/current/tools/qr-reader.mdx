## How to use

1. Select or drag & drop an image file containing a QR code.
2. If analysis is successful, the embedded text content will be displayed.
3. If the content is a URL, you can open the page directly with the button.
4. Extracted text can be easily saved to your clipboard using the copy button.

:::info Security
🔒 Uploaded images are never sent to any server. All analysis is done locally within your browser to ensure privacy.
:::
