## How to use

1. Enter two dates you want to compare.
2. The tool supports various formats like slash-separated (2026/01/01), natural language (January 1st, 2026), and Unix timestamps.
3. You can click the calendar icon to pick a date from the native picker.
4. Check "Always compare with current time" to fix the first date to the current moment and see real-time updates.
5. Results are shown in "Days/Hours/Minutes/Seconds" and "Total Seconds" formats, both copyable to your clipboard.

:::info Privacy
🔒 All date data is processed entirely within your browser and is never sent to any server.
:::
