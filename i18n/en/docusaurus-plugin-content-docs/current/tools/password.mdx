## How to use

1. Specify the length of the password and the types of characters to use (symbols, uppercase, lowercase, etc.).
2. Changing the settings will automatically generate a new password.
3. Use the "Copy" button to copy individual passwords, or the "Save" button to download the generated list as a text file.
4. Your password generation settings are reflected in the URL, so you can bookmark it to quickly generate passwords under the same conditions next time.

:::info Security
🔒 All processing is done entirely within your browser. Passwords and settings are never sent to any server.
:::
