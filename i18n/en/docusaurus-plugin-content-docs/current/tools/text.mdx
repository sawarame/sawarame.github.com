## How to use

1. Enter text in the input area on the left and click the "Save" button or press Enter (depending on your settings) to save it.
2. Saved text will be added to the list on the right and stored in your browser's local storage.
3. You can pin frequently used text to the top using the pin icon.
4. Utilize saved texts using the copy or bulk download features.

:::info Security
🔒 All data is stored within your browser. It is never sent to any server.
:::
