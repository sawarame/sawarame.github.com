# Feature Overview

Explore the organization and customization features available in TabOrigami.

## Flexible Organization Styles

Choose the best categorization style to match your current workflow:

- **Auto**: Let the AI automatically determine the best categories.
- **Task Focus**: Group tabs by ongoing tasks or projects.
- **Work-Life**: Clearly separate business-related tabs from personal ones.
- **Triage**: Delete unnecessary tabs.

## Custom Prompt Feature

You can enter custom prompts to provide free-text instructions to the AI. This allows you to achieve your preferred tab grouping.

1. Click "Add Custom Prompt" in the popup.
2. Enter your instructions for the AI in the text field, and click the "Save" button.
3. Go back to the main screen, choose any organization style, and run it. The suggested groupings will now reflect your custom prompt instructions.

:::tip
You can also provide multiple instructions at the same time in the custom prompt. For example:
```
- If there are pages with the same URL, keep only one and delete the rest.
- Set the tab group names as emojis.
```
:::

## Undo Organization (History Feature)

If you are not satisfied with the organization results or want to revert to a previous state, you can restore them using the following steps:

1. Click "Undo Organization" in the popup.
2. A list of up to 10 previous pre-execution states will be displayed. Select the point in history you wish to revert to.
3. Your tabs will be restored to that state.

## Exclude Pinned Tabs

This feature protects important tabs you always keep open (pinned tabs).
Please turn on "Exclude pinned tabs from organization" in the options screen. This ensures that pinned tabs will always be excluded from the organization.
