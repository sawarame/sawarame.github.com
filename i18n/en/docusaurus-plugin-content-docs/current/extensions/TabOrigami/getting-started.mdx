# Getting Started

Here are the basic steps to start using TabOrigami.

## 1. Get a Gemini API Key

TabOrigami uses a "Bring Your Own Key" (BYOK) approach, which means users need to obtain and configure their own Google Gemini API key. Please follow these steps to get your API key:

1. Access [Google AI Studio](https://aistudio.google.com/app/apikey) and sign in with your Google account.
2. Click the "Create API key" button under "API keys" to generate a new key.
3. Enter any key name and select a project. You can choose an existing project or create a new one.
4. Click the "Create key" button. Once the API key is generated, **copy the key**.

## 2. Set Your API Key

Follow these steps to configure the obtained API key in TabOrigami:

1. Click the Chrome extension icon for TabOrigami to open the popup.
2. Click the settings icon ⚙️ in the top right corner and register your **API Key**.
3. Change the language (Japanese / English) or the Gemini model to use if necessary.

## 3. Organize Your Tabs

Once the setup is complete, try organizing your tabs.

1. On the popup screen, select an "Origami Style" (organization style) that fits your current workflow.
2. The AI will analyze your tabs and display a preview of its suggested groupings.
3. Review the suggestions. You can drag and drop tabs to move them around or uncheck the checkboxes to exclude them from the organization.
4. You can also freely edit the tab group names and colors.
5. Click the **"Execute"** button to apply the groups to your Chrome tabs.
