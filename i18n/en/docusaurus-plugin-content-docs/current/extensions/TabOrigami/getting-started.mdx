# Getting Started

Here are the basic steps to start using TabOrigami.

## 1. Set Your API Key

TabOrigami uses a "Bring Your Own Key" (BYOK) approach, meaning you will use your own Gemini API key.

1. Click the extension icon to open the TabOrigami popup.
2. Click the settings icon at the bottom of the screen and register your **Gemini API Key** (required).
3. If necessary, select your preferred language (Japanese / English) and the Gemini model to use (e.g., gemini-2.5-flash).

## 2. Organize Your Tabs

Once the setup is complete, try organizing your tabs.

1. On the popup screen, select an "Origami Style" (organization style) that fits your current workflow.
2. The AI will analyze your tabs and display a preview of its suggested groupings.
3. Review the suggestions. You can drag and drop tabs to move them around or use checkboxes to fine-tune which tabs are included.
4. Click the **"Execute"** button to apply the groups to your Chrome tabs. The Smart Cleanup feature will also automatically close any unnecessary "New Tab" pages.

## 3. Undo Organization (History Feature)

If you don't like the result or want to revert to a previous state, you can restore your tabs using the following steps:

1. Click the "History" icon at the top of the popup screen.
2. You will see a list of up to 10 previous pre-execution states. Select the point in history you wish to revert to.
3. Your tab configuration will be restored to that state.
