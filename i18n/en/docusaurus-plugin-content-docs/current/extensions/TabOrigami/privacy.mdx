# TabOrigami Privacy Policy

TabOrigami (hereinafter referred to as "this extension") prioritizes your privacy and strives to provide a secure environment for all users. This Privacy Policy explains how personal information and data are handled within this extension.

## 1. Information Collected and Its Purpose

This extension uses the following information to organize your tabs. **All of this information is processed locally on your device or transmitted solely to Google's official Gemini API. It is never sent to or stored on external third-party servers, including those of the developer.**

* **Open Tab Information (Titles, URLs, and Page Content)**:
  This information is collected and sent to the Gemini API to perform advanced AI categorization and organization.
* **Gemini API Key**:
  This extension adopts a "Bring Your Own Key" (BYOK) approach. The API key you provide is stored securely only in your device's local storage (browser sync storage) and is used exclusively for communicating with Google's official Gemini API.

## 2. External Data Transmission

When executing the tab organization feature, this extension directly sends the collected tab information and your API key to the "Gemini API" provided by Google LLC.

* The transmitted data is processed in accordance with Google's API Terms of Service and Privacy Policy.
* **No data is ever sent to servers managed by the developer (Sawarame) or any other third-party servers.**

## 3. Data Storage

* User settings (such as your API key, selected language, and model name) and your past tab organization history (up to 10 entries) are saved in your browser's local environment (e.g., `chrome.storage.local`).
* All of this data will be completely removed if you uninstall the extension.

## 4. Security

This extension is built in compliance with "Manifest V3," the latest security standard for browser extensions. It requests only the minimum necessary permissions and performs API requests through secure data communication (HTTPS).

## 5. Changes to the Privacy Policy

This Privacy Policy may be updated without prior notice due to the addition of new features or changes in laws and regulations. In the event of significant changes, we will notify you via the extension's update notes or the official webpage.

## 6. Contact Us

If you have bug reports, feedback, feature requests, or any questions regarding privacy, please contact us via [GitHub Issues](https://github.com/sawarame/TabOrigami/issues).
