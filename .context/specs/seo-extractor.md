# 15. SEO情報（メタデータ）抽出ツール (`docs/tools/seo-extractor.mdx`) 仕様書

ブラウザ上で完結して動作し、貼り付けられたHTMLソースコードからSEO関連のメタデータ（タイトル、説明文、キーワード、favicon、OGP画像など）を抽出し、表示・プレビュー・エクスポートするツールです。

## 主な機能・仕様
* **完全ローカル解析方式 (CORS制限なし)**:
  * **HTMLコピペ解析**: ユーザーがコピーしたHTMLソースコードをテキストエリアに貼り付けることで、外部サーバーに一切通信せず完全にブラウザ内だけで安全に解析を行います。
  * **プライバシー保護**: 社内向けの開発中システムや、機密データ・個人情報を含むページのコードであっても情報流出の心配がなく、オフライン環境でも動作します。
* **抽出データ項目**:
  - **基本情報**: タイトル（`<title>`）、説明文（`description`）、キーワード（`keywords`）
  - **OGP情報**: `og:title`, `og:description`, `og:image`, `og:url`, `og:type`, `og:site_name`
  - **Twitter Card**: `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image`
  - **リンク/その他**: favicon画像、`canonical` URL、ロボット設定（`robots`）
* **リアルタイム文字数検証**:
  - タイトル（推奨: 25〜35文字）および説明文（推奨: 80〜120文字）の文字数をカウントし、SEO的な適合度を視覚的にカラーインジケーター（プログレスバー）で表示。
* **ビジュアルプレビュー**:
  - **Google検索結果風プレビュー**: 検索エンジンで表示された際のタイトル、favicon、サイト名、説明文のデスクトップ検索モックを表示。
  - **SNSシェア風プレビュー**: OGP画像、タイトル、ドメイン名を含んだカード型シェア表示のモックを表示。
* **データ操作**:
  - 各メタデータのワンクリックコピー機能。
  - 抽出した全データをJSON形式でファイル保存（エクスポート）可能。デフォルトファイル名は `seo_metadata_YYYYMMDDHHMMSS.json`。
