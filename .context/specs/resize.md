# 7. 画像軽量化・クロップツール (`src/pages/resize.tsx`) 仕様書

ブラウザ上で画像をクロップ（切り抜き）し、WebPなどの次世代形式に変換・軽量化するツールです。

## 主な機能・仕様
* **完全ローカル処理**:
  * 全ての画像処理（圧縮、切り抜き、変換）はブラウザのCanvas APIおよびクライアントサイド向けライブラリを使用して実行されます。画像データが外部サーバーへ送信されることは一切ありません。
* **クロップ（切り抜き）機能**:
  * **自由な範囲選択**: マウス操作による直感的な切り抜き範囲の指定が可能。
  * **比率プリセット**: 1:1 (正方形), 8:5 (1.6:1), 1.91:1 (OGP), 16:9 (X ポスト), 3:1 (X ヘッダー) 等の主要なアスペクト比にワンクリックで固定可能。
  * **高解像度維持**: プレビューの表示サイズに関わらず、元画像の解像度を維持したまま切り抜きを行います。
* **最適化・変換設定**:
  * **出力フォーマット**: WebP（推奨）, JPEG, PNG, オリジナル（同形式）から選択可能。
    * ※WebP変換非対応のブラウザでは自動的にJPEGが推奨として選択されます。
  * **画質 (Quality)**: 0.1〜1.0までスライダーで調整可能。
  * **最大幅指定**: プリセット（1280px, 1080px等）のほか、任意のピクセル数を手入力で指定可能。縦長の画像であっても「幅」がこの数値に収まるようにリサイズされます。
  * **EXIF保持**: JPEG出力時のみ、元のメタデータ（カメラ情報等）を維持するか選択可能。
* **比較プレビュー (Before/After)**:
  * 圧縮前と後の画像をスライダーで左右に動かして比較。
  * **ズーム・パン機能**: 最大5倍までの拡大およびドラッグによる移動に対応し、細部の劣化を詳細に確認可能。
* **バッチ処理**:
  * 複数画像の一括アップロードと一括変換に対応。
  * 変換後の画像は個別に、または `JSZip` を用いて一括でZIPダウンロードが可能。
* **高精度リサイズ**: 比率指定クロップとサイズ制限（最大幅）を併用した際、計算誤差による1pxのズレを防ぎ、指定した目標幅に基づいた正確な解像度で出力します。
