# テキスト文字コード変換 (Text Character Code Converter)

## 1. 概要
テキストファイルの文字コードを自動判別し、指定した文字コード（UTF-8, Shift_JIS, EUC-JP等）に変換してダウンロードできるツール。

## 2. ターゲットユーザー
- 文字化けしたテキストファイルを修復したいユーザー
- 特定のシステム（古いExcelやレガシーシステム等）向けに文字コードを変換したいユーザー
- ファイルをアップロードせずにブラウザ内で安全に処理したいユーザー

## 3. 機能要件

### A. ファイル入力
- ドラッグ＆ドロップ、ファイル選択、またはペーストによるテキスト入力に対応。
- ファイルだけでなく、クリップボード上のテキストを直接ペーストして変換することも可能。
- ファイルサイズ制限：数MB程度を想定（ブラウザのメモリ制限内）。

### B. 文字コード自動判別
- アップロードされたファイルのバイナリを解析し、文字コードを自動推測する。
- 対応予定コード：UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP, UTF-16等。

### C. プレビュー
- 判別された文字コードと、そのコードでデコードしたテキストの冒頭数KBを表示。
- 文字化けしている場合に備え、ユーザーが手動で「現在の文字コード」を再選択できる機能。

### D. 文字コード変換・ダウンロード
- ユーザーが変換先の文字コードを選択。
- 変換後のデータをBlobとして生成し、ダウンロード。
- ファイル名は `(元ファイル名)_(文字コード).txt` またはユーザー指定。

## 4. 非機能要件
- **完全ローカル処理**: サーバーへファイルを送信せず、ブラウザ内（Web Worker推奨だが小規模ならメインスレッドでも可）で処理。
- **プライバシー保護**: 入力されたテキスト内容はどこにも記録・送信しない。
- **多言語対応**: 日本語と英語の UI を提供。

## 5. 使用ライブラリ（候補）
- `encoding-japanese`: 日本語環境で一般的に使用される文字コードの判別・変換に優れる。

## 6. UI設計案
- 上部：ファイルアップロードエリア。
- 中央左：現在の設定（自動判別結果、手動変更）。
- 中央右：プレビュー（デコード結果）。
- 下部：変換設定（出力文字コード選択）とダウンロードボタン。
