# テキスト文字コード・改行コード変換 (Text Encoding & Line Break Converter)

## 1. 概要
テキストファイルの文字コードおよび改行コードを自動判別し、指定した形式（UTF-8, Shift_JIS等 / LF, CRLF等）に変換してダウンロードできるツール。

## 2. ターゲットユーザー
- 文字化けしたテキストファイルを修復したいユーザー
- 特定のシステム（Windows/Unix等）向けに改行コードを統一したいユーザー
- ファイルをアップロードせずにブラウザ内で安全に処理したいユーザー

## 3. 機能要件

### A. ファイル入力
- ドラッグ＆ドロップ、ファイル選択、またはペーストによるテキスト入力に対応。
- ファイルだけでなく、クリップボード上のテキストを直接ペーストして変換することも可能。

### B. 文字コード・改行コード自動判別
- アップロードされたファイルを解析し、文字コードと改行コードを自動推測する。
- 対応文字コード：UTF-8, Shift_JIS, EUC-JP, ISO-2022-JP, UTF-16等。
- 対応改行コード：LF (Unix), CRLF (Windows), CR (Classic Mac)。

### C. プレビュー
- 判別された文字コードと改行コード、およびデコードしたテキストの冒頭を表示。
- 設定が正しくない場合に備え、ユーザーが手動で現在の設定を修正できる機能。

### D. 変換・ダウンロード
- ユーザーが変換先の文字コードおよび改行コードを選択。
- 一度 Unicode 文字列として処理することで、文字コードと改行コードの同時変換を実現。
- ファイル名は `(元ファイル名)_(文字コード).txt` またはユーザー指定。

## 4. 非機能要件
- **完全ローカル処理**: サーバーへファイルを送信せず、ブラウザ内で完結。
- **URL同期**: 変換設定（ターゲット文字コード、改行コード）をURLパラメータで共有可能。
- **多言語対応**: 日本語と英語の UI を提供。

## 5. 使用ライブラリ
- `encoding-japanese`: 日本語環境の文字コード判別・変換に使用。

## 6. UI設計
- 上部：ファイルアップロードエリア。
- 中央：プレビューエリア（メタ情報含む）。
- 下部：変換設定エリア（文字コード、改行コードの選択）とダウンロードボタン。
