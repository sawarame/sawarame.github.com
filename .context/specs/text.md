# 2. テキスト保存場所 (`docs/tools/text.mdx`) 仕様書

一時的なメモや複数行のテキストをブラウザの `localStorage` に保存し、即座に呼び出しおよび管理ができる簡易スクラッチパッドです。

## 主な機能・仕様
* **テキスト入力と保存設定**:
  * 複数行のテキスト入力エリアを提供
  * 「エンターで保存」の挙動切り替え（ON時は `Enter` 単体で保存、OFF時は `Ctrl(Cmd) + Enter` で保存）
* **データ管理 (`localStorage`)**:
  * 保存したテキストは作成日時の記録と共に `SavedTexts` として永続化される
  * 最新のメモが最も見やすいように、一覧UIの上部から表示される
  * **ピン留め機能:** 重要なテキストをピン留めし、常に一覧の最上部に表示させることが可能（ピン留め状態も `SavedTexts` に保存される）
* **アクション機能**:
  * 各メモの個別コピーおよび個別削除、ピン留めの切り替え
  * 保存済みテキストの一括コピーと一括削除
  * 保存済みテキストすべてを `.txt` 拡張子でダウンロード（各テキストは日時ヘッダを付与してリスト化）
