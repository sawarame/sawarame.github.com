# マークダウンPDF変換ツール (`docs/tools/markdown-pdf.mdx`) 仕様書

アップロードされたマークダウンファイル（`.md`）や直接入力したマークダウンテキストを、ブラウザ上でPDFに変換してダウンロードできるツールです。
テキストデータとして選択・検索可能なPDFをクライアントサイドで生成します。

## 主な機能・仕様

* **入力方法**:
  * **ファイルアップロード**: マークダウンファイル（`.md`）のドラッグ＆ドロップ、ファイル選択ダイアログ、またはクリップボードからの貼り付け（Paste）による読み込みに対応。
  * **エディタ入力**: 画面上のテキストエリアに直接マークダウンを記述可能。プレビュー機能付き。
    * **PDF自動プレビュー機能**: 「PDFプレビュー」タブを選択したタイミングで、マークダウンの内容が変更されているか、まだPDFが生成されていない場合に自動でPDFを生成して表示します。フォントの読み込みやPDF生成の間はローディングインジケーターが表示されます。これにより、余分な生成処理を抑えて入力のパフォーマンスを最大化しています。
* **PDF変換オプション**:
  * **フォントサイズ**: 小（10pt）、中（12pt）、大（14pt）から選択可能。
  * **ページ余白**: 狭い（10mm）、標準（20mm）、広い（30mm）から選択可能。
  * **用紙サイズ**: A4、Letterから選択可能。
  * **用紙の向き**: 縦（Portrait）、横（Landscape）から選択可能。
  * **ヘッダー/フッター**: ページ番号の追加（表示/非表示）を設定可能。
* **日本語対応 (フォント)**:
  * クライアントサイドでの日本語文字化けを防ぐため、オープンソースの日本語フォント（Sawarabi Gothic）をローカルに配置し、クライアントサイドで動的に読み込み、jsPDFに登録して使用します。
* **URLパラメータ同期**:
  * オプションの設定値は、リアルタイムでURLのクエリパラメータに反映されます。
  * 対応パラメータ:
    - `fs`: フォントサイズ (`small`/`medium`/`large`)
    - `mg`: 余白 (`narrow`/`normal`/`wide`)
    - `sz`: 用紙サイズ (`a4`/`letter`)
    - `or`: 用紙の向き (`portrait`/`landscape`)
    - `pn`: ページ番号を表示するか (`true`/`false`)
* **データの保存・永続化**:
  * 入力されたマークダウンテキストは `localStorage` に一時保存され、ブラウザのリロード時にも保持されます。
* **高度なマークダウンレンダリング**:
  - **リスト (List)**: 任意のネスト階層に対応したインデント描画。
  - **引用 (Blockquote)**: 複数ページにまたがる場合でも、左側の縦線が途切れずに連続して描画されます。
  - **コードブロック (Code Block)**: ページをまたいでも背景のグレー色塗りつぶしが崩れずに描画されます。
  - **テーブル (Table)**: セル内のテキスト自動折り返しに対応し、ページをまたぐ場合は次のページの上部にテーブルヘッダーを自動再描画します。
  - **カラー絵文字 (Color Emojis) 対応**:
    * jsPDFは標準フォントで絵文字をレンダリングできないため、サロゲートペア対応の処理で絵文字を検出し、Canvas APIを用いて一時的にOSネイティブフォント（Apple Color Emoji, Segoe UI Emoji等）で描画。それを画像（PNG）としてPDFにインライン埋め込みします。
    * **頭切れ防止**: 絵文字の上下左右（特に頭部）が切れるのを防ぐため、キャンバスの描画領域をフォントサイズの1.4倍（余白比率20%）に広げて描画し、PDF生成時に座標とサイズをオフセット調整して埋め込んでいます。
* **アクション機能**:
  * PDFダウンロード: 指定した設定値に基づいてPDFを生成・ダウンロードします。デフォルトファイル名：`markdown_[YYYYMMDDHHMMSS].pdf`
  * HTMLダウンロード: パースされたマークダウンに見やすいスタイル（GitHub風）を適用したHTMLファイルを生成し、ダウンロードします。デフォルトファイル名：`markdown_[YYYYMMDDHHMMSS].html`
  * サンプル挿入: 初めてのユーザー向けにマークダウンのサンプルを挿入するボタン。
  * コピー＆クリア: テキストのワンクリックコピー、クリア機能。

## セキュリティ・プライバシー
* すべての処理はユーザーのブラウザ内（クライアントサイド）で完結します。
* アップロードされたファイルや入力されたテキストが外部サーバーに送信されることはありません。
