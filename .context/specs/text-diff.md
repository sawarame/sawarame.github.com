# テキスト差分（Diff）比較ツール 仕様書

## 1. 概要
2つのテキストデータまたはファイルを比較し、その差分を視覚的に表示するツール。複数のファイルをアップロードし、その中から任意の2つを選択して比較することができる。

## 2. 機能要件

### A. ファイル入力
- **複数ファイルアップロード**: ドラッグ＆ドロップ、ファイル選択、または貼り付け（Paste）による複数ファイルの取り込み。
- **テキスト直接入力**: フォームへの直接入力による比較（任意だが、使い勝手向上のため検討）。
- **文字コード自動判別**: `encoding-japanese` を使用し、Shift_JIS 等のファイルも文字化けせずに読み込む。

### B. ファイル管理
- **アップロード済みリスト**: アップロードされたファイルの一覧を表示。
- **比較対象の選択**: リストの中から比較元（Original）と比較先（Modified）の2つを選択。
- **削除機能**: リストからファイルを削除。

### C. 差分表示
- **ビューモード**: 
  - **Split (2画面)**: 左右に並べて比較。
  - **Unified (1画面)**: 1つの画面内で行ごとに比較。
- **オプション**:
  - 空白文字の無視（Ignore Whitespace）。
  - 大文字小文字の無視。
- **シンタックスハイライト**: テキストの内容に応じたハイライト（可能であれば）。

### D. ローカル処理
- 全ての比較処理はブラウザ内で行い、サーバーへの送信は一切行わない。

## 3. UI/UX デザイン

### レイアウト
1.  **アップロードエリア**: 既存の「テキスト文字コード変換」と同等のデザイン。
2.  **ファイルリスト**: アップロードされたファイルがカードまたはリスト形式で並び、ラジオボタン等で「比較元」「比較先」を指定。
3.  **コントロール**: ビューモードの切り替え、オプション設定。
4.  **ディフビューアー**: 画面下部に大きく表示。

### デザイン・トーン
- [DESIGN.md](../../DESIGN.md) に準拠。
- MUI v6 コンポーネントを使用。

## 4. 技術スタック
- **差分ロジック**: `diff` (jsdiff)
- **UI コンポーネント**: `react-diff-viewer-continued` (GitHubライクな表示のため)
- **文字コード処理**: `encoding-japanese`

## 5. 多言語対応 (i18n)
- 日本語・英語の二ヶ国語対応。
- `i18n/en/` 配下の JSON を更新。

## 6. 保存・共有
- **URL同期**: ビューモードや無視設定をクエリパラメータに保持。
- **エクスポート**: (将来的に) 差分結果の HTML/PNG 出力。
