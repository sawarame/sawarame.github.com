# 8. PDF変換・編集 (`src/pages/pdf-editor.tsx`) 仕様書

PDFファイルを画像（PNG, JPEG, WebP）に変換したり、複数のPDFから必要なページだけを抽出・結合して新しいPDFを作成できるツールです。

## 主な機能・仕様
* **完全ローカル処理**:
  * `pdfjs-dist` および `pdf-lib` を使用し、ブラウザ上でPDFのレンダリングや編集を行います。画像やPDFデータが外部サーバーに送信されることはありません。
* **ページ選択とプレビュー**:
  * 全ページ、または特定のページのみを選択して変換・抽出の対象にすることが可能です。
  * **高画質プレビュー機能**: 各ページのサムネイル（虫眼鏡ボタン）をクリックすると、ブラウザ画面に合わせた大きなサイズで内容を確認できるプレビューモーダルが開きます。
  * モーダル内では、「次へ」「戻る」によるページ送りや、変換対象としての選択／解除を直接行うことができます。
* **柔軟なエクスポート機能**:
  * **画像として書き出す**:
    * **出力フォーマット**: PNG, JPEG, WebPから選択可能。
    * **最大横幅指定**: 任意の横幅（px）に収まるようにリサイズして出力します（アスペクト比は維持）。
    * 選択されたページが1枚の場合は直接画像ファイルとして、複数ページの場合は `JSZip` を用いてすべての画像をZIP形式でまとめてダウンロードします。
  * **PDFとして書き出す**:
    * アップロードされた1つ、または複数のPDFファイルから選択したページのみを抽出し、それらを結合した新しいPDFファイルとしてダウンロードします。
* **使い方の案内**:
  * 初めて利用するユーザー向けに、画面内に操作手順とセキュリティ（完全ローカル処理）に関するガイドを常時表示。
