# 1. パスワードジェネレーター (`docs/tools/password.tsx`) 仕様書

指定した条件に基づいて、セキュアなランダムパスワードをクライアントサイド（ブラウザ上）で生成するツールです。

## 主な機能・仕様
* **設定項目**:
  * **パスワードの長さ:** 1〜128文字まで指定可能
  * **作成数:** 一度に生成するパスワードの数（1〜20個まで）
  * **使用しない文字:** 指定した文字を生成されるパスワードから除外する
  * **大文字・小文字の除外設定:** プリセットとしてアルファベットの大文字、または小文字を一括して使用しないように設定可能
  * **使用可能な記号:** 指定した記号のみを使用可能にする（空欄の場合は全記号を使用）
  * **文字の重複制限:** 「同じ文字を使用する」のON/OFFにより重複を許容するか設定
  * **記号の使用:** 「記号を使用する」の全般的なON/OFF
* **URLパラメータ同期**:
  * 各設定値はリアルタイムでURLのクエリパラメータに反映され、設定状態をURLとして保存・共有が可能
  * 対応パラメータ: 
    * `len`: 長さ
    * `count`: 作成数
    * `exclude`: 除外文字（個別入力）
    * `no_upper`: 大文字を除外 (true/false)
    * `no_lower`: 小文字を除外 (true/false)
    * `symbols`: 使用可能記号
    * `same`: 重複制限
    * `use_sym`: 記号使用
  * パラメータが含まれるURLでアクセスした際、その値を初期値として読み込み、自動的にパスワードを生成する
* **アクション機能**:
  * 各パスワードごとのクリップボードへの単体コピー機能
  * 全ての生成結果をテキストファイル（`.txt`）として保存するダウンロード機能
  * ダウンロードファイル名のフォーマット: `[現在日時(YYYYMMDDHHMMSS)].txt`
* **外部サイトへの埋め込み機能**:
  * `src/pages/embed/password.tsx` により、設定パネル（SettingsCard）を非表示にし、生成されたパスワード（ResultCard）のみを表示する埋め込み専用ページ（`/embed/password`）を提供。
  * `PasswordGenerator` に `embedded` プロパティを追加し、真の場合はレイアウトを埋め込み用に最適化。
  * `EmbedCodeSection` コンポーネントは、URLパラメータ（パスワード設定）を動的に引き継ぎ、埋め込み用iframeコードおよびURLを生成する。
